Exercício 1
Crie uma função anônima que some dois valores caso a soma seja menor ou igual a 0 e multiplica os dois valores caso contrário.

```
myFunc = lambda x, y: x + y if x + y <= 0 else x * y

print(myFunc(-3,2))
```
